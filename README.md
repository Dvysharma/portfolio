# portfolio

Personal portfolio of Divyanshu Sharma — Data Analyst, Business Analyst, AI Enthusiast, and Technology Consultant.

## About

This is a Next.js portfolio website built with TypeScript, Tailwind CSS, Framer Motion, and React Three Fiber. It showcases projects, experience, skills, research, and contact details.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Deployment

This project can be deployed directly to Vercel as a Next.js app. Use the default build settings and add any required environment variables.
